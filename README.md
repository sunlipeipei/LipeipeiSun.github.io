# Lipeipei Sun's Personal Portfolio

Welcome to the repository for my personal portfolio website.

## Technology Stack

The portfolio is built using:
- **[React](https://react.dev/)**: For component-based UI development
- **[Vite](https://vitejs.dev/)**: For lightning-fast development, bundling, and hot module replacement
- **[React Router](https://reactrouter.com/)**: For seamless and dynamic client-side navigation
- **[React Bootstrap](https://react-bootstrap.netlify.app/) & [Bootstrap](https://getbootstrap.com/)**: For responsive layouts and core UI components
- **[Lucide React](https://lucide.dev/)**: For beautiful, consistent iconography

## Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sunlipeipei/LipeipeiSun.github.io.git
   cd LipeipeiSun.github.io
   ```

2. **Install dependencies:**
   Make sure you have [Node.js](https://nodejs.org/) installed, then run:
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   The site will be available locally (usually at `http://localhost:5173/`).

## Building for Production

To create an optimized production build, run:
```bash
npm run build
```
This generates the static assets inside the `dist` directory, which can then be served statically or deployed to GitHub Pages.
